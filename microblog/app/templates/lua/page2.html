3. Введение в Функции Lua

Функции в Lua позволяют вам повторно использовать код и делать его более организованным. В этом уроке мы рассмотрим, как создавать и использовать функции в Roblox.

3.1 Функции в Lua

Функции – это блоки кода, которые выполняют определенную задачу. Вы можете вызывать функцию многократно в разных частях вашей программы. Давайте создадим простую функцию в Lua:


-- Function declaration
function greeting()
    print("Hello, player!")
end

-- Function call
greeting()

Эта функция выводит "Hello, player!" в консоль. Вы можете вызывать эту функцию столько раз, сколько нужно.

3.2 Задание 1: Функция с параметром

Давайте изменим функцию, чтобы она принимала параметр – имя игрока:


-- Function with a parameter
function greeting(name)
    print("Hello, " .. name .. "!")
end

-- Function call with an argument
greeting("Alice")

Теперь функция будет выводить приветствие с именем игрока. Попробуйте вызвать функцию с разными именами.

Задание 2: Функция для создания объектов

Создадим функцию, которая будет создавать объекты в Workspace. Эта функция примет параметры для определения свойств объекта:


-- Function to create a part
function createPart(name, size, position)
    local part = Instance.new("Part")
    part.Name = name
    part.Size = size
    part.Position = position
    part.Anchored = true
    part.Parent = game.Workspace
end

-- Function call to create different parts
createPart("Part1", Vector3.new(4, 1, 2), Vector3.new(0, 10, 0))
createPart("Part2", Vector3.new(2, 2, 2), Vector3.new(5, 15, 5))

Эта функция позволяет создать части с разными свойствами, вызывая ее с разными аргументами.
