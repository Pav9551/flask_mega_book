3. Введение в функции Python

Функции позволяют вам повторно использовать код и сделать его более организованным. В этом уроке мы рассмотрим, как создавать и использовать функции в Python.

3.1 Функции в Python

Функции – это блоки кода, которые выполняют определенную задачу. Вы можете вызывать функцию многократно в разных частях вашей программы. Давайте создадим простую функцию в Python:


# Определение функции
def greeting():
    print("Привет, игрок!")

# Вызов функции
greeting()

Эта функция выводит сообщение "Привет, игрок!" на экран. Вы можете вызывать эту функцию столько раз, сколько нужно.

3.2 Задание 1: Функция с параметром

Давайте изменим функцию, чтобы она принимала параметр – имя игрока:


# Функция с параметром
def greeting(name):
    print(f"Привет, {name}!")

# Вызов функции с аргументом
greeting("Алиса")

Теперь функция будет выводить приветствие с именем игрока. Попробуйте вызвать функцию с разными именами.

Задание 2: Функция для расчета

Создадим функцию, которая будет принимать два числа и возвращать их сумму:


# Функция для расчета суммы
def add_numbers(a, b):
    return a + b

# Вызов функции с аргументами
result = add_numbers(3, 5)
print(result)

Эта функция принимает два числа и возвращает их сумму. Попробуйте вызвать эту функцию с разными числами.
