1. Введение в Python для начинающих

Python — это простой и мощный язык программирования, который используется для создания игр, веб-сайтов и даже научных проектов. Сегодня мы начнем изучать его основы!

Первое, что нужно знать о Python — это то, что он очень легко читается. Давайте разберем несколько простых примеров.

2.1 Первые шаги с Python

Для начала давайте напишем простую программу, которая выведет сообщение на экран. Это делается с помощью команды print().


# Ваш первый Python-код!
print("Привет, мир!")

Запустите эту программу, и вы увидите сообщение "Привет, мир!" на экране.

2.2 Задание 1: Придумай свое сообщение

Измените программу так, чтобы она выводила ваше имя или любимую фразу вместо "Привет, мир!". Для этого просто замените текст внутри кавычек.


# Пример:
print("Меня зовут Алиса, и я учусь программировать!")

Теперь программа выведет ваше сообщение! Попробуйте разные варианты.

Задание 2: Знакомство с ветвлениями

Ветвления позволяют выполнять определенные участки кода в зависимости от условий. Например:


import random
# Генерация случайной температуры
t = random.randint(15, 30)
print("Температура:", t, " градусов")
# Проверка условий и вывод рекомендаций
if t > 25:
    print("Температура слишком высокая! Включаем кондиционер.")
elif t < 18:
    print("Температура слишком низкая! Включаем обогреватель.")
else:
    print("Температура комфортная, ничего менять не нужно.")

Можете разнообразить ветвления не забывайте про отступы

2.4 Знакомство с циклами

Циклы позволяют повторять действия несколько раз. Например, с помощью цикла мы можем 5 раз напечатать одну и ту же фразу:


# Пример цикла
for i in range(5):
    print("Программировать - это круто!")

Эта программа напечатает сообщение "Программировать - это круто!" 5 раз. Попробуйте изменить количество повторений.

Задание 3: Таблица умножения

Напишите программу, которая с помощью цикла выводит таблицу умножения для числа 3:


# Таблица умножения для 3
for i in range(1, 11):
    print("3 *", i, "=", 3 * i)

Когда вы запустите программу, она выведет таблицу умножения для числа 3.
