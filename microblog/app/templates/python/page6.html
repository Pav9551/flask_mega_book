1. Загрузка и вывод карт из файла

Сегодня мы создадим программу на Python, которая загружает карты из текстового файла и выводит их. Это полезно для отображения данных в консольном приложении в структурированном виде, например, для игрового процесса или визуализации данных.

2.1 Описание функции загрузки карт

Функция load_maps_from_file считывает текстовый файл и разбивает его содержание на отдельные карты, где каждая карта отделена пустой строкой. Карта представляется списком строк.

2.2 Настройка функции

Функция print_map выводит карту, принимая список строк, содержащих разделенные пробелами значения.


def load_maps_from_file(filename):
    maps = []
    current_map = []
    with open(filename, 'r', encoding='utf-8') as file:
        for line in file:
            line = line.strip()  # Удаляем пробелы и символ новой строки
            if len(line) == 0 and current_map:
                maps.append(current_map)
                current_map = []
            elif line:
                current_map.append(line)
        if current_map:
            maps.append(current_map)
    return maps

def print_map(unicode_map):
    for line in unicode_map:
        for code in line.split():
            print(code, end=' ')
        print()  # Новая строка после каждой строки карты

def main():
    filename = 'maps_unicode.txt'
    try:
        maps = load_maps_from_file(filename)
        for i, map_ in enumerate(maps):
            print(f"Map {i + 1}:")
            print_map(map_)
            print()
    except FileNotFoundError:
        print(f"Файл {filename} не найден.")
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()

Задание 1: Основная логика выполнения

Основная функция main инициирует загрузку карт из файла, а затем каждая карта отображается в консоли. Это полезно для визуального представления структуры данных.
